Hablemos de tu proyecto

¿Por qué elegir el Motor SOLO CH SPEED?

Ofrecemos automatización completa para portones corredizos utilizando el Motor SOLO CH SPEED de 400kg, diseñado para un rendimiento superior y durabilidad. Este motor incluye un microcontrolador ARM de 32 bits con procesamiento de 46MHz, memoria para hasta 250 controles programados, ajustes rápidos con trimpots y desaceleración electrónica independiente. Ideal para residencias, comercios o industrias, garantizamos una instalación sencilla, personalizada y soporte continuo para un funcionamiento óptimo.

Características Destacadas

Microcontrolador ARM de 32 bits

Procesamiento de 46MHz para un control preciso y eficiente del motor, con última generación de tecnología.

Memoria Interna para 250 Controles

Capacidad para programar hasta 250 botones o controles directamente en el microcontrolador.

Ajustes Rápidos con Trimpots

Principales ajustes por medio de trimpots para una configuración fácil y sin complicaciones.

Desaceleración Electrónica Independiente

Ajuste de rampa de llegada para apertura y cierre, asegurando movimientos suaves y seguros.

Alimentación Full Range 90-240 VAC

Compatible con una amplia gama de voltajes para mayor versatilidad en cualquier instalación.

Sistema de Seguridad contra Alta Temperatura

Protección automática en caso de sobrecalentamiento o necesidad de movimiento manual.
